# SF2568-parallel-computing-for-large-scale-problems

## Homework 1

Run in Windows:

```bash
docker build -t sf2568-hw1 . && docker run -v %cd%\data:/etc/data sf2568-hw1
```

Linux/Mac:

```bash
docker build -t sf2568-hw1 . && docker run -v ${PWD}/data:/etc/data sf2568-hw1
```
